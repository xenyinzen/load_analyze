Do_chart relies on thess libs:

libgd.so.2
libz.so.1
libpng12.so.0
libfreetype.so.6
libjpeg.so.62
libm.so.6
libc.so.6
libfontconfig.so.1
/lib/ld.so.1
libexpat.so.1
libpthread.so.0

In fact, only if you install package of libgd2-noxpm in debian, all above
dependencies relation should be satisfied. 

In our loonux 1.1.2 or above, you can use this tool directly.
