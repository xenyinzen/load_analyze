Sysrec Readme
2009-3-13

tangg@lemote.com
=======================================================================

This tool is to record the system load information using 'top'.

if parameter 1 is omitted, choose the default file name 'top_result.txt'
in the directory '../record_result'. 

if parameter 2 is omitted, use the default 1 seconds.

if parameter 3 is omiited, use the default 'infinite' times.

usage:

./sysrec [filename.txt] [interval] [times]
